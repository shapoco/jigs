# 作業自撮り用スマホスタンド

![](./images/cover.jpg)

![](./images/cad_ss.png)

小物や手元の作業を撮りたいけど三脚を用意するほどでもないようなときに気軽に使えるスマホスタンドです。

- スマホを7種類の角度で固定できます。
- 底面に 1/4-20 UNC のネジ穴があり、三脚等の雲台に固定できます。
